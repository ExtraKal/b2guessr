BTDB2 Map Spotter

Folders:
- index.html -> main site layout
- assets/css/styles.css -> all styling
- assets/js/maps.js -> map metadata, filenames, HOM flags, candidate clue points
- assets/js/app.js -> game logic, help modal, scoring, filtering, blueprint rendering
- assets/images/maps -> all real map images used by the game

Notes:
- Game modes: All Maps and Hall of Masters Only
- Difficulties: Beginner, Intermediate, Advanced, Expert, CHIMPS
- Max score per round: 100
- Wrong map = 0 for the round
- Blueprint view is generated from the same map image with a blue filter
